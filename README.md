# ClassSchedule

一个浏览器中运行的可视化课程表工具，用于快速编排一周课程、导入导出课表数据，并保存课程表图片。

## 功能

- 拖拽排课：从左侧“待排课程”拖动课程到右侧周课程表。
- 课程管理：添加新课程，双击左侧课程名称可编辑。
- 自定义时长：不同课程类型可设置不同课时，同一课程保持相同课时。
- 自动配色：不同课程自动分配不同颜色，便于区分。
- 删除课节：点击课程表中的课节后确认删除。
- 自动保存：课程配置和已排课节会保存到浏览器本地，下次打开自动恢复。
- 导入导出：支持将当前课程表和待排课程导出为 JSON，也可以从 JSON 恢复。
- 保存图片：将右侧课程表区域导出为 PNG 图片。
- 周视图：默认显示周一到周日，时间范围为 8:30 到 22:00，按半小时显示时间刻度。

## 使用方法

直接点击开始使用：

[在线预览](https://classschedule.pages.dev/)

直接用浏览器打开 `课程表.html` 即可使用。页面会通过 CDN 加载以下前端库：

- FullCalendar：课程表视图和拖拽排课
- html2canvas：将课程表保存为图片

添加课程时需要填写该课程的时长。左侧课程卡片会显示课程名称和时长，之后拖入课程表的新课节会自动使用对应课程的固定时长。双击左侧课程可以修改名称和时长，已排入课程表的同名课程会同步更新。

课程名称、课程时长和已排课节会自动保存到当前浏览器的本地存储中。每次添加、编辑、拖拽或删除课程后都会自动写入浏览器本地保存；刷新页面、重新打开页面或更新 HTML 文件后，页面会优先恢复这个浏览器里的自动保存。只有当前浏览器还没有自动保存数据时，才会尝试加载服务器同目录下的 `课程表.json` 作为初始数据。

如果需要让页面启动时自动读取同目录下的 `课程表.json`，建议在项目目录启动一个本地静态服务器后访问页面：

```bash
python3 -m http.server 8000
```

然后打开：

```text
http://localhost:8000/课程表.html
```

## 数据文件

页面会自动保存当前浏览器中的课程表。点击“导出JSON”还可以生成 `课程表.json` 作为备份或迁移文件；点击“导入JSON”会用文件内容恢复课表，并立即写入自动保存。导出的文件包含：

- `calendarEvents`：已经排入课程表的课节，每节课只保存星期、开始时间和结束时间，不绑定具体日期
- `availableCourses`：左侧待排课程列表
- `courseColorMap`：课程名称与颜色的映射
- `courseDurationMap`：课程名称与固定时长的映射

`课程表.json` 可能包含个人课程安排，默认不会提交到 Git 仓库。需要分享课表时，可以手动发送导出的 JSON 或 PNG 文件。

## 适用场景

- 学校或培训机构临时编排课程表
- 家庭学习计划安排
- 个人周学习任务可视化
- 快速生成可分享的课程表图片

## 隐私说明

项目本身不包含后端服务，也不会主动上传课表数据。课程表数据保存在浏览器当前页面状态或用户手动导出的本地文件中。
